# CYA-2 README

Project: Independent Exploration and Implementation of Quantum Computing (Grover's Algorithm)

## Description of the Chosen Adventure

The project name is: Independent Exploration and Implementation of Quantum Computing (Grover's Algorithm). In this project, I chose to pursue an independent technical upskilling project focusing on Quantum Computing. Specifically, I conducted an in-depth study of Grover's Algorithm, transitioning from theoretical mathematical foundations to practical implementation. I utilized IBM's open-source framework, Qiskit, to simulate quantum circuits and applied the algorithm to solve practical constraints like small-scale Sudoku and Boolean Satisfiability (SAT) problems.

## Summary of the Deliverable

The tangible deliverable for this project consists of:

- A Comprehensive Technical Report (Grovers_Algorithm_Report.md): A detailed research document that explains the core steps, geometric interpretation (amplitude amplification), and performance analysis of Grover's Algorithm.

- Implementation Code / Notebook: Code implementations utilizing the qiskit library. The code demonstrates:

    - Building an Oracle and a Grover diffusion operator.

    - Applying the algorithm to solve a $2 \times 2$ Sudoku matrix using multi-controlled-Toffoli gates.

    - Using the PhaseOracle class to solve a 3-SAT problem based on a DIMACS file.
